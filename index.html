Projects

CURRENT - A Novel Approach to IVF Treatment using Deep Learning

In this project, we aim to improve the accuracy the embryo selection process of IVF treatments using deep learning methods as well as mathematical analysis methods on segmentation maps obtained from embryos.

Evaluation of the performance of nnUNet on fatal head challenge

This project aims to evaluate the performance of nnUNet on the fatal head challenge dataset.

Modelling State Transitions in Gene Regulatory Network

This project aims to create mathematical models for gene regulatory networks and simulate their state transitions. We aim to encode the state transitions into latent space to explore their relations. Also it will be published as a modular python package to create customizable grn models

Facial Landmark Detection on Cleftlips with Data Synthesis via Image Inpainting

This work aims to develop a machine learning model that can detect the facial landmarks of cleft lip patients that can be later adapted into a deep learning model that can annotate cleft lip surgical facial landmarks. We explored different data generation methods such as photo editing and facial inpainting using the CTSDG model
